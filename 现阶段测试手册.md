### B-2 合成日志 → 数据集转换（#38）

方案一还没有真机日志，先造一条合成 HDF5（模拟 RunLogger 输出格式）：

```bash
python - <<'EOF'
import numpy as np, h5py
from pathlib import Path
Path("il/logs_il").mkdir(exist_ok=True)
n = 1250                               # 10s @125Hz
with h5py.File("il/logs_il/synthetic_run.h5", "w") as f:
    t = np.linspace(0, 10, n)
    f["t"] = t.reshape(-1, 1)
    f["state"] = np.where(t < 4, 0, np.where(t < 7, 1, 2)).reshape(-1, 1)
    tcp = np.tile([0.45, 0.0, 0.25, 0.0, 3.1416, 0.0], (n, 1))
    tcp[:, 2] -= np.linspace(0, 0.03, n)          # 模拟缓慢下插
    f["tcp"] = tcp
    f["target"] = tcp + np.random.normal(0, 1e-4, tcp.shape)
    w = np.zeros((n, 6)); w[:, 2] = -8 * (t > 4)  # 接触后 -8N
    f["wrench_h"] = w + np.random.normal(0, 0.2, w.shape)
    f.attrs["state_codes"] = str({"guarded_move": 0, "search": 1, "insert": 2})
print("synthetic_run.h5 written")
EOF

python il/learning/convert_h5.py --h5 "il/logs_il/synthetic_run.h5" --repo dualarm_smoke
```

**通过标准**：打印 `synthetic_run.h5 → 250 帧`（1250/5 下采）、`转换完成: 1 条 episode`；
`il/logs_il/datasets/dualarm_smoke/` 下生成 `meta/` `data/` 目录结构。
建议顺手多转 3–5 条（重跑生成脚本改随机种子），训练冒烟数据更像样。

### B-3 训练冒烟（#40）

**通过标准**：
